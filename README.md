# ZenChain GM DApp 🚀

A simple decentralized application (dApp) on **ZenChain Testnet**.  
Users can send "GM" transactions and see the **total number of GMs** and the **latest GM messages**.

---

## ✨ Features
- 🔗 Connect wallet with **MetaMask**
- 💬 Send GM transactions to the smart contract
- 📊 Display **Total GMs** and **Last GMs**
- 🎨 Simple UI with custom CSS
- 🌐 Deployable on **GitHub Pages**
